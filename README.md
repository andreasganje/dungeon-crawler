# Dungeon crawler

A first person grid based dungeon crawler engine script for RPG games written in PHP.
Provides rendering of dungeon graphics and auto mapping.

* Details and samples: https://ganje.de/blog_technik_first-person-grid-based-dungeon-crawler.html#titel
* Demo: https://demo.ganje.de/move-3d_dungeon/
